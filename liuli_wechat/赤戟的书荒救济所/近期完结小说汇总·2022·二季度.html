1.都市·娱乐文·行业文·商战文


都市幻想


《灵魂画手》无籽甜瓜，味道很正的克苏鲁都市文·惊悚向·画家职业文·高开中走，主角是一名神经质艺术家，SAN值起伏如坐过山车，每天在疯狂边缘反复横跳，背景是现代都市调查员的世界观，主线日常就是被卷入一件件特殊的调查事件，然后将经历的事件绘就画作，通过创作这些让人精神污染的画作收割SAN值，用收割的SAN值给自己的能力加点。需要注意的是，虽然味道很正，也引入了不少爽点，但整体来看，还是有些爽度不足，比较小众，行文中存在少量bug和打脸情节，干白硬，特别是“诺贝尔数学奖”事件，中期画佛剧情开始中走，出现不少差评的声音，特别是进入日本篇后口碑热度迎来了下降，争议较大。首发起点，142万字，完本，干草 / 粮草

《无敌从拳法大成开始》三生愚， 强者主角·刃牙风格的高武都市文·高开低走，主角意志坚定，唯武唯拳，一意武道，没有太多男女私情，冷酷似钢，行事干净利落，光明正大但又灵活多变。是那种有进取心的强者主角，虽然初始体质普通，拳法二流，但强者之心，永不屈服，越战越勇，善于在战斗中吸收对手的武道精华不断成长。争议在于剧情过于紧凑，主角一路杀伐无双，武戏太多，文戏不足，而且力量体系并非单纯的都市国术文，之后会混入克苏鲁、神话、灵气复苏等元素，从反馈来看，建议追到地球篇结束即可，之后的剧情一言难尽。首发起点，234万字，完结，干草 / 粮草-

都市狗粮

《我的恋爱指数要满仓》鱼鱼快动，日本校园日常·二次元恋爱后宫文·年上控，高仿版《我加载了恋爱游戏》，主角魂穿于日本旧贵族，前身父母私奔，家族中边缘化，向青梅表白被拒后，主角穿越而来，主打轻松愉快的恋爱日常，金手指是恋爱股市，妹子的股价随着对主角的好感度上涨下跌，然后主角操纵股市赚点数买系统商店的点数和技能。开篇一般，之后渐入佳境，感情戏写的非常好，典型日轻的人设，个性鲜明足够有趣，没啥胃药，就是单纯的师生快乐修罗场，首发起点，156万字，完本，干草 / 粮草-

都市产业

《1991从芯开始》三分糊涂，罕见的芯片产业文，海芯首席架构师的苏远山重生回1991年。从EDA开始布局，在瓦森纳协定之前抢占先机，各种被神兽穷追猛打，整体专业性强，但剧情平淡无毒点，细节有点刻意，主角年龄偏小。反馈来看，普遍带着善意的鼓励，涉及各行各业，尤其是芯片，光刻机等崛起，对芯片通讯手机感兴趣当做发展史看看就行，对小说情节的欣慰以及对现实的意难平大概就是这类都市重生产业文的最大看点了，期待不宜过高，首发起点，270万字，完本，干草 / 干粮+


2.科幻·末日·无限流


《我们生活在南京》天瑞说符，年度科幻文·开篇让人惊艳的末日拯救类精品，2019年的男主是个玩短波通信高三学生，用父亲的ICOM725短波电台，意外与来自2040年高等文明入侵的末日后幸存者女主用电台完成通话，故事就此展开。以现实城市南京为故事开始的场所，加强了合理性和代入感。不管是文笔还是氛围的营造都很有味道，独自顽强生活在末世半夏，应该是迄今为止天瑞笔下最鲜活的角色。故事围绕时光慢递三定律展开，探讨时间的慢递规则，并且依据这个规则构建了最后两个大高潮，构思精巧，故事惊艳，介绍了很多无线电台的知识，硬核又充满畅想的浪漫，男主现实线和女主末日线的双线对比非常新颖，有《末世第十年》那味，再加上科幻元素，画面感十足。需要注意的是，开放式结局，到最后男女主也没有相遇，不是传统意义上的HE（或者说：“对于人类来说是HE，对于爱情是BE”），据说所有的内容伏笔都在楔子里提到了，在情节发展中一一回收，最后的伏笔也在终章揭幕。首发起点，40万字，完本，干草 / 仙草


《寄生之子》群星观测，脑洞佳作·温馨有趣科幻文，主角是外星人和他的两个同族，分别寄生在脑死亡的小男孩，一只猫和一只鸡身上，故事双线展开，主线是寄居地球的主角鸡飞猫跳的咸鱼生活，日常努力解决身边的外星异常事件，暗线则是和绿脚鹈鹕相关的时空线（过去线），勾连长辈们的故事，扑朔迷离，遍布玄机。整个故事在“平静日常与奇妙异常之间的来回横跳”。主线和日常搞笑生草结合得自然不突兀，想象力很强，文字上是那种淡淡的叙事风格，温馨中带着现实的残酷，平稳和荒诞的日常中缓慢推进主线，孩童的视角则带来趣味和生活化的气息，加之各种脑洞和有趣的小故事，有点皮皮鲁版本的黑衣人的感觉，从书友@御禾 的反馈来看：“最后主角团HE，挫败了反派的阴谋，全文草灰蛇线，伏线千里，很多前面的情节后面都有呼应的地方，大结局解开了最大的疑惑，把全文都串联了起来”，首发晋江文学城，177万字，完本，干草 / 仙草

《长夜余火》爱潜水的乌贼，公路片风格的废土风情画·风格另类的救世文，主角是个想要“拯救全人类”的“精神病人”，毕业后加入危险度极高的旧世界毁灭原因调查小组，开始地面的公路探险，通过主角四人组的视角，一步步揭开新旧世界的面纱，展现一幅荒诞却自洽的废土风情画。需要注意的是，争议极大，故意打散节奏、不直接描写心理活动、克制煽情描写，全程抑制主角，这就导致一个后果：世界观方面依然构建得很好，但缺乏代入感，情感上也很难引起共鸣。主线进度慢，事件之间的驱动力不足，很多事件完结后留下一堆谜团，所幸现在完本了，可以一口气看完，结尾比较仓促，主角最终实现了自己的初心，有头有尾，就是比较刀，具体不剧透了，从阅读感受来看，这本书可以算得上一种“游戏攻略实况”吧。算是非常反网文主流套路的一本作品，如果看腻了传统网文套路，可以来试试本书，风格独特，适合猎奇爱好者。首发起点，236万字，完本，干草 / 粮草+ （可能是某些童鞋的仙草吧，不过我不是特别有感觉）

《非正常世界修仙法则》沁纸花青，荒诞风仙侠·AI工具人反抗的短篇故事·神经病风格。《心魔》作者出品，一本实验性质的先锋作品，怪诞荒谬的沙雕文，乃是集天才修仙，顶级智斗，家庭伦理，轮回世界，科幻未来，宇宙星战，最终灭世龟零的一本奇文！用了三十章非常荒诞生艹震惊读者的剧情，揭开西部世界一般的故事主线。作者的智斗和编排剧情相当有一手，运用了不少小说套路梗，弱智吧金句，快手土味鬼畜梗，并以一种出人意料的方式化用到了剧情之中，喜剧效果爆炸，但荒诞搞笑下满是细思极恐，精神污染效果极强。如果你喜欢《小明修仙传》和《李姐的超能力世界》，那本书值得一看，对了，结合本章说你讲拥有更多的乐趣。首发起点，11万字，完本，毒草 / 仙草 （整活儿文，某种意义上又一本网文奇书诞生了）

《女寝大逃亡[无限]》火茶，以寝室为单位的无限流·无感情戏，讲的是全世界大学生突然被拉入了"游戏"，女主和3个舍友一起想办法"毕业"，整个过程中对人物的刻画感觉还是不错的，大体上是无限流，设定合理，内容不算血腥暴力，但在剧情上面狠下了功夫，最后he通关。脑洞很大设定新颖，公众号里中好评为主，据说书籍评论区有很多魔障言论，男生如果要看，别点开评论区。感谢书友@微笑 @图南矢北 @C 的认真推评和反馈，首发晋江文学城，92万字，完本，干草 / 粮草-


3.仙侠·武侠·玄幻


《顶级气运，悄悄修炼千年》任我笑，签到类苟道流仙侠·小白文，看书名知道是本小白文，点进去也确实是本小白文，但意外还能看，作为套路水文，苟中苟的修炼打卡流，可以看做鬼谷八荒的同人，主角特别苟，一直苟山上埋头修炼，800多岁了，突破大乘成散仙了，愣是没出过宗门一步，总的来说，水至极的苟道白爽文，把苟道贯彻到底的无敌流，打架全靠秒，文笔和剧情的确都很白，不合理之处甚多，经常凑字数，但是胜在流畅，不影响整体的阅读感觉，从头到尾一如既往地苟，没整幺蛾子。首发起点，245万字，完本，水草 / 粮草- ps：如果是喜欢《神秘之劫》的苟道仙侠副本，本书就值得一试。

《江湖风云第一刀》枯空散人，古龙风武侠诸天流，《江湖位面小人物》作者出品，主角是个14岁的穿越者少年，虽然是穿越者但不知道剧情，心性不错，杀伐果断。以武侠小说中的名刀串联起来，用书友@二十六日 的话说，武侠味非常冲，文风、情节和对白都很有古龙风味，当然本质上还是爽文，不用担心虐主。争议在于：“用古龙的手法改写金庸意识流，导致画风有点儿怪，部分副本存在争议，但可以跳过不影响”，这书放出来的时候口碑很不错，但中途因为一些争议就没再关注，直到完本之后口碑热度都很不错，才重新回头审视，发现累计反馈近60条，好评为主，之前低估这书了，喜欢无限武侠流的书友可以试试，首发起点，132万字，完本，干草 / 粮草

《佛爷棍下不留生》造化斋主，有着豪侠韵味的“江湖”文·正气十足，小说背景是灵气污染畸变的末世世界，灵机被污染，靠灵格修行，力量体系借鉴了诡秘的序列魔药和扮演法，与魔药类似，灵格同样有污染的特性。亮点在于主角的侠僧人设，用书友的话来说，“主角是真正意义上的侠，除恶务尽而不走极端，即使成为魔佛的if线最终也为众生度过末日而选择了牺牲，本质上还是悲情反英雄”，另一个值得称道的是主角没有犯教条主义错误，种田时做出的改革符合生产力和世界架构。争议在于，画风轻快，剧情略显老套，嘴炮不断，故事剧情线偏慢，更新也慢，中后期更新拖拖拉拉，结局仓促，用书友@陈鹏 的话来说：“主线剧情基本完整，不能说烂尾，但结局确实略仓促。作者说大概砍了大概100万字的支线剧情”。首发刺猬猫APP，152万字，完本，干草 / 粮草


4.历史·军事·架空


《江户旅人》秽多非人，切入点有趣的日本冷门历史文，主角穿越德川幕府末期1841年的江户，二十年后就将迎来明治维新，初期身份是小和尚，疑似岛津家私生子，师傅死后还俗，辅佐自己的发小做官，同时获得了晋身之阶，暂时为幕府效力养望，卷入幕府与外藩之争。相比前作，少了一些自嗨，延续之前资料扎实的风格，行文流畅，剧情推进有条不紊，各种小细节充实，日常细节写的富有生活气息。争议在于，非主流网络爽文的写作方式，科普性强，故事性弱，偶尔还要文青一下，各种时代术语太多，中后期越来越白开水，推演合理性也一般。首发起点，216万字，完本，干草 / 粮草- ps：目前在更新番外

《二分之一剧透》天泽时若，女帝题材的亏成首富流变种·迪化类架空争霸文，女主带着一半的剧透评论穿到书里，被系统要求做个大昏君败光家业，一心想成为昏君的主角，被书评论区的反向玩梗剧透，误解了人物评价，硬是把昏君做成了千古一帝。争议在于：“政斗整体偏简单小儿科，作者试图写一个男女平等继承法的架空封建王朝末期的社会，但是字里行间都是写男硬说女的感觉，整体智商不高的样子”，不过迪化流都有点儿这种，所以期待不宜过高。公众号里的口碑热度还不错，作为一本晋江女频书也有近20条留言，好评为主，题材爱好者可以试试，首发晋江文学城，64万字，完本，干草 / 粮草


5.西方奇幻


《神级狩魔人》隐约点，少见的巫师3同人·慢热考究·有游戏内味·非爽文，主角重生到北境战争前的农夫之子，之后拜师雷索，成为蛇派猎魔人四处冒险，金手指是系统加点，但是升级很慢，主打剧情，估计后续发展路线是集各大学派所长，重现猎魔人传承。剧情颇有游戏内味，争议在于升级感不明显，慢热不够爽，前期毒点不少，作为一个成长型主角，开篇稚嫩，心智不成熟，“强行降智”，后期主角战力飙升，收尾急促，很多坑没填，但瑕不掩瑜，勉强算有头有尾，整体算是写出了猎魔人在很多问题上进退两难的感觉，不但流血又流泪，还成了过街老鼠，阅感是比较憋屈的，所以不建议爽文爱好者入。首发起点，252万字，完本，毒草 / 粮草 ps：有书友认为超凡世界的超凡者被平民人人喊打，很离谱不合理，但这就是游戏里的设定，所以不喜勿入吧，没必要战设定哈~


6.游戏·竞技

《我真没针对法爷》十年一键，独行侠·单机刷级流·游戏异界文，轻松向稳健无敌流·行文爽白诙谐，文笔老道，剧情流畅不拖沓，文风欢快不轻佻，人设有趣双商在线，主角深谙老阴比之苟道，明白自己要做什么，该怎么做，目标明确，行动干净利落，字里行间骚中带梗~作者精心设定了一套针对法师的职业体系，刷特定的怪，获得特定专长的设定，设定党看起来很带感。刷刷刷的风范，真的有一种玩游戏的感觉。争议点在于，一，单机刷怪，人物互动很少；二，金手指太大，获得东西太容易，数据化战力涨得飞快。三，完结仓促，前段时间作者生病，回来复更，把支线砍了一些，就直接完结了，对了，这书一次性看多了容易腻歪，适合养一段看一段，优书评分5.3/10，首发起点，272万字，完本，干草 / 粮草-

和平鲨鱼《从山海经复苏开始》，留言区涌现·姑获鸟风格的无限流，据说是卖盘的狐狸的马甲，凭依的是山海经里吃了能治痔疮的虎蛟，优书评分5.4/10，别期望过高，这作者最近的几本书都太监了，入坑需谨慎。感谢@东玉 @雨霁拾遗 @Mr.yz 等书友的认真推评和反馈，累计反馈7条，3条中差评，首发起点，45万字

三生三笑《粤食记》，罕见都市美食文·科普向·双女主，主要是介绍广东美食，丰富的粤菜文化和传统知识，像舌尖上中国一样找美食，不太像网文风格，剧情一般。双姐妹女主，作者还是有东西的，比一些作者直接百度写美食文好多了。感谢书友@🐴 华 @returnor 的认真推评和反馈，累积反馈3条，2好1差：“粤菜版的中华小当家，各种华丽炫技但与现实毫不想干”，首发起点，87万字



2. 近期推书反馈汇总~（完本书和新书，以及近期口碑翻红的养肥书）


近期新书里，据说刺猬猫《女皇陛下请自重》作者跑起点证道的《古神在低语》，都市诡异·异能复苏文，宅味较重，力量体系的名字有点low，但公众号里口碑尚可，感兴趣的可以试试。


《赛博魔术师的日常》作者一木啊的新书，《怪谈作者拖更日记》，怪谈类副本流，融入文字合成类脑洞，怪谈副本，动物园规则怪谈等等等，脑洞和氛围感到位，都还挺有趣，也有小十条反馈了。


养肥书里，有两本大神作品值得关注：


一本是录事参军的《我的老婆是执政官》，魂穿上世纪四五十年代纽约为背景的异时空律师，开篇极限逆转被绞刑的命运，节奏很紧凑，累积反馈18条，15条好评：“题材较为新颖，后宫熟女，剧情干净利落，看的痛快。”，3条中差评：“合理性有瑕疵，作为一个法学生，东拼西凑术语也不太规范，看着主角装逼确实难顶，但是剧情还算流畅”。


另一本是写过《深空之下》《深空之流浪舰队》《全球迈入神话时代》的最终永恒新书，灵气复苏·幕后黑手·游戏影响现实的《异维度游戏》值得关注，这书五月份新品尝鲜里推过的，不过当时反馈相对较少，就没有再关注，但从六月份开始，好评热度持续不断，目前也累积了近20条推书留言，只有一条中差评。
